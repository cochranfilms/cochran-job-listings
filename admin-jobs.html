📋 How This Works

Workflow:

  1. Add/Edit Jobs → Use this admin panel to create job listings
  2. Save Changes → Click "💾 Save Changes" to download jobs-data.json
  3. Upload to GitHub → Upload the file to your repository
  4. Auto-Update → GitHub Action automatically updates your live site
  5. Google Sheet → Handles all applicant submissions automatically

Note: The Google Sheet handles applicant data separately - this admin panel only manages job listings.

GitHub Action: Once you upload jobs-data.json to GitHub, it will automatically update your live site at collaborate.cochranfilms.com
