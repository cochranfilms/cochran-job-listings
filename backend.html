📋 How Job Management Works

Workflow:

  1. Add/Edit Jobs → Use the admin panel to create job listings
  2. Export JSON → Download the updated jobs-data.json file
  3. Update Website → Replace the jobs-data.json file on your server
  4. Google Sheet → Handles all applicant submissions automatically

Note: The Google Sheet handles applicant data separately - the admin panel only manages job listings.

Job Management Benefits

  • ✅ Centralized job listing management
  • ✅ Automatic JSON export for website updates
  • ✅ Seamless integration with Google Sheets
  • ✅ Real-time status tracking
  • ✅ Managed dropdown options for consistency

👥 How Creator Management Works

🚀 Fully Automated Creator Management:

  1. Add Creators → Use dropdown selectors for all fields (roles, locations, rates, dates)
  2. Auto-Job Creation → Primary job automatically created from creator details
  3. Auto-Approval → When you save a new creator, they automatically receive a job acceptance email
  4. Manual Approval → For existing creators, click the "✅ Approve" button to send job acceptance email
  5. Additional Jobs → Assign extra jobs to creators using managed dropdown options
  6. Contract Signing → Creators sign contracts directly on contract.html and set their own password
  7. Automatic Updates → Password and contract status automatically update in freelancers.json
  8. Email Notifications → You get instant email alerts when contracts are signed
  9. Real-time Dashboard → See password status and contract progress in real-time
  10. Managed Options → All dropdown options are managed through the dropdown management system

📧 Email Setup: Configure EmailJS credentials in contract.html to receive signing notifications.

🎯 Benefits: No password conflicts! Creators set their own passwords during contract signing, and everything updates automatically with managed dropdown options.

📧 Email Notification Setup

To receive email notifications when contracts are signed, configure EmailJS in contract.html:

Setup Instructions:

  1. Sign up for EmailJS: Go to emailjs.com and create a free account
  2. Create Email Service: Connect your email provider (Gmail, Outlook, etc.)
  3. Create Email Template: Design your contract signing notification email
  4. Get Credentials: Copy your Public Key, Service ID, and Template ID
  5. Update contract.html: Replace the placeholder values in EMAILJS_CONFIG

💡 Template Variables Available:
freelancer_name, freelancer_email, role, location, project_start, rate, signature_date, signed_timestamp, effective_date
