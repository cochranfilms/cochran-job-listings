📧 Email Notification Setup

                
To receive email notifications when contracts are signed, configure EmailJS in contract.html:

                

                    
Setup Instructions:

                    
    
                        
  1. Sign up for EmailJS: Go to emailjs.com and create a free account
    2. 
                        
  2. Create Email Service: Connect your email provider (Gmail, Outlook, etc.)
    3. 
                        
  3. Create Email Template: Design your contract signing notification email
    4. 
                        
  4. Get Credentials: Copy your Public Key, Service ID, and Template ID
    5. 
                        
  5. Update contract.html: Replace the placeholder values in EMAILJS_CONFIG
    6. 
                    

                    

                        

                            💡 Template Variables Available:

                            freelancer_name, freelancer_email, role, location, project_start, rate, signature_date, signed_timestamp, effective_date

☁️ Google Drive Sync Setup

                

                    

                        Cross-Device Contract Sync: Your signed contracts will be stored in Google Drive for access from any browser/device.
                    

                    
                    

                        
📋 Setup Instructions:

                        
    
                            
  1. Go to Google Cloud Console: console.cloud.google.com
    2. 
                            
  2. Create New Project: Name it "Cochran Films Contracts"
    3. 
                            
  3. Enable Google Drive API: Search for "Google Drive API" and enable it
    4. 
                            
  4. Create Credentials:
                                
      
                                    
    • Go to "Credentials" → "Create Credentials" → "API Key"
      • 
                                    
    • Copy your API Key
      • 
                                    
    • Create "OAuth 2.0 Client ID" for web application
      • 
                                    
    • Add your domain to "Authorized JavaScript origins"
      • 
                                    
    • Copy your Client ID
      • 
                                
    
                            
    5. 
                            
  5. Update Configuration: Replace the placeholder values in GOOGLE_DRIVE_CONFIG
    6. 
                        

                    

                    
                    

                        

                            💡 Benefits: Once configured, contracts will automatically sync across all your devices. 
                            The file "cochran-films-signed-contracts.json" will be created in your Google Drive.
